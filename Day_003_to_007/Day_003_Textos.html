Bitácora de un Futuro Programador

Día 3: Dominando el Esqueleto de la Web

Hoy oficialmente arranqué a escribir código en HTML5. Al principio pensé que era un lenguaje de programación, pero ya aprendí que es un lenguaje de maquetación,encargado de darle estructura visual a una página.

Mis Objetivos de esta Semana

Para no distraerme, voy a seguir un cronograma estricto enfocado en un solo tema por día:

  • Estructurar correctamente documentos HTML.
  • Aprender a meter enlaces y navegación maquetada.
  • Dominar el uso de imágenes, audios y videos.
  • Aprender a organizar información usando tablas de datos.

Mi Combo de Comandos Favorito

Cada vez que termino una práctica, uso estos pasos ordenados para actualizar mi GitHub:

  1. Ejecutar git status para ver qué cambió.
  2. Usar git add textos.html para preparar mi archivo.
  3. Escribir git commit -m "feat: agregar practica de textos" para capturar el archivo.
  4. Lanzar un git push origin main para subirlo a mi repositorio.
